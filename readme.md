
**Proyecto: Página Web Informativa Básica**

**Descripción:**
Este proyecto consiste en la creación de una página web sencilla, desarrollada con tecnologías básicas de desarrollo web. La página contiene información simple y clara, enfocada en mostrar contenido básico sin funcionalidades avanzadas.

**Instalacion**
Buscar el en el GitHub el repositorio a consultar los puedes decargar o crear una copya desde la terminal para incializar


**Tecnologías utilizadas:**
**HTML:** Para la estructura de la página.
**CSS:** Para el diseño y estilos visuales.

**Características:**
Información básica.
Diseño simple.
Estructura clara y ordenada.
Proyecto introductorio o de práctica.

**Objetivo:**
El objetivo de este proyecto es practicar y reforzar conocimientos básicos en el desarrollo web utilizando HTML y CSS.
